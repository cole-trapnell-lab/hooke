Introduction

Single-cell transcriptome sequencing (sc-RNA-seq) experiments allow us to track cell populations between conditions and experiments. The Hooke package provides a toolkit for analyzing changes in single-cell populations.

Under construction

Hooke is currently in the beta phase of its development. This means there are likely bugs and performance issues that will need to be addressed. We are working hard towards a stable release, but please be patient while Hooke is under construction.

The documentation on this page is also still under construction. Not all features currently implemented have been completely documented.
