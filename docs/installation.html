Installing Hooke

Required software

Hooke runs in the R statistical computing environment. You will need R version 4.2.0 or higher, Monocle 3 version 1.3.1 or higher, torch version 0.9.0 or higher, and PLNmodels version 1.0.0-9020 or higher.

To install Monocle 3, follow the on-line Monocle 3 installation instructions.

Next, install torch and PLNmodels using the command

{% highlight R %} install_packages(c('torch', 'PLNmodels')) {% endhighlight %}

If you wish to install the develop branch of hooke, execute:

{% highlight R %} devtools::install_github('cole-trapnell-lab/hooke', ref="develop") {% endhighlight %} Previous Next
