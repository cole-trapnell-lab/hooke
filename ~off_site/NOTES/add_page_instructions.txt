Unfortunately adding a top level docs page is currently a big pain... here's how:

Make a new html named something to do with your new page, i.e. alignment.html, save it in the docs folder

Add the following text to the top of the page, lines you need to change are described after ####:

Notes:
  o  the permalink is set to the file root name
  o  the 'description:' describes the page
  o  the toc for file has the 'class="active"' attribute in the anchor tag
  o  the main content area includes a list item with a brief description for the page
  o  the toc entry of the file must be added to all other pages tocs but
     without the 'class=active' attribute
