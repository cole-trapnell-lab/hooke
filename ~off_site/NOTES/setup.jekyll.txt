Steps for deploying the website

1. follow the instructions for installing jekyll on you system
URL: https://jekyllrb.com/docs/installation/#guides

2. install node.js
from package installer available at:
https://nodejs.org/en/

3. Build the site:
bundle exec jekyll build --incremental --watch

4. Serve the site (in a second terminal window)
bundle exec jekyll serve --watch

5. Load the site in chrome:
http://127.0.0.1:4000/hooke/
